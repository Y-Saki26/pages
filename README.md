# Sakishita Yuki Web Pages

Sakishita Yuki の個人サイト

* Jekyll テーマ: ['Minimal'](https://github.com/pages-themes/minimal) を使用
* MathJax: `$ $`  でインライン数式，`$$ $$` でアウトライン数式
* [Jekyll Sitemap Generator Plugin](https://github.com/jekyll/jekyll-sitemap) で "./sitemap.xml" にサイトマップを生成
* フッターにトップへ戻るボタン，最終更新日を表示
